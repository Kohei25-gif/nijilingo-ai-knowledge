# 🍎 NijiLingo 検証方法

## 📝 テスト文章（必ずこの3つを使う）

```
1. その服素敵だね
2. 遅れてごめん、電車が止まってた
3. 明日の会議の資料を送ってもらえる？
```

---

## ✅ 検証項目と確認ポイント

### 検証①: 敬語レベル（バグ1）

**テスト文:** その服素敵だね / 遅れてごめん、電車が止まってた

**やること:**
1. business または formal（超丁寧）トーンを選ぶ
2. スライダーをLv0 → Lv1 → Lv2 → Lv3 と動かす

**確認ポイント:**
- 日本語（逆翻訳）が段階的に丁寧になってるか
- 英語も段階的に変化してるか

**NGパターン:** Lv2がLv1より敬語弱い / 英語が全く変わらない
**OKパターン:** Lv0 < Lv1 < Lv2 < Lv3 と段階的に丁寧

---

### 検証②: 待つと結果が変わる（バグ2）

**テスト文:** どれでもOK

**やること:**
1. どれかトーンを押す
2. 翻訳結果をチェック
3. **そのまま5-10秒待つ**
4. 結果が変わってないか確認

**確認ポイント:**
- 待機中に結果が勝手に変わらないか
- 何回か操作しないと翻訳が変わらない現象がないか

**NGパターン:** 待ってたら違う翻訳に変わる / 操作しないと翻訳が反映されない
**OKパターン:** 待っても同じ結果のまま / 操作したらすぐ反映

---

### 検証③: modality_class（バグ3）

**テスト文:** 明日の会議の資料を送ってもらえる？

**やること:**
1. **casualトーン**を選ぶ
2. スライダーを**Lv3（100%）**にする
3. 翻訳結果を確認

**確認ポイント:**
- 「送ってもらえる？」（依頼）が「送ってるね？」（確認）に変わってないか
- 英語が "Can you..." / "Could you..." のまま維持されてるか

**NGパターン:** 依頼文 → 確認文に変化（"Can you" → "Are you"）
**OKパターン:** 依頼文のまま維持

---

## 🔧 追加検証: ロード速度

**確認ポイント:**
- トーン押してから翻訳表示までの時間
- 体感で遅いと感じないか

---

## 📸 スクショの撮り方

1. **各レベルごとに1枚**（Lv0, Lv1, Lv2, Lv3）
2. **翻訳結果と逆翻訳が両方見える状態**で撮る
3. **トーン名とレベルが画面に表示されてる**状態で撮る
